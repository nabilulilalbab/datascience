1. BUSINESS UNDERSTANDING

1.1 Latar Belakang Masalah

Industri telekomunikasi merupakan salah satu sektor utama di negara-negara berkembang. Kemajuan teknologi dan meningkatnya jumlah operator telah meningkatkan tingkat persaingan di pasar (Ahmad et al., 2019).

Fakta Kritis: Biaya akuisisi pelanggan baru dapat mencapai 6x lebih tinggi daripada biaya retensi pelanggan yang sudah ada.

Customer churn (perpindahan pelanggan dari satu provider ke provider lain) menjadi masalah kritis yang berdampak langsung pada pendapatan perusahaan. Perusahaan perlu mengidentifikasi faktor-faktor yang menyebabkan pelanggan churn dan mengambil tindakan preventif.

1.2 Tujuan Bisnis

  1. Mengembangkan model prediksi churn dengan akurasi tinggi (target AUC ≥ 90%)
  2. Mengurangi tingkat churn melalui intervensi proaktif
  3. Meningkatkan retensi pelanggan sebagai strategi revenue terbaik
  4. Mengoptimalkan alokasi sumber daya pemasaran

3.1 Konsep Statistik Deskriptif

Berdasarkan Vanlalhriati dan Singh (2015), statistik deskriptif berfokus pada merangkum karakteristik data.

Measures of Central Tendency

MetodeFormulaKegunaan
Mean x̄ = (1/n) Σxᵢ Rata-rata tagihan, durasi panggilan
Median Nilai tengah Mengurangi efek outlier
Mode Nilai terbanyak Paket yang paling populer

Measures of Spread/Variance

MetodeFormulaKegunaan
Variance σ² = (1/n) Σ(xᵢ - x̄)² Variasi penggunaan layanan
Std Deviation σ = √σ² Deviasi standar durasi
IQR Q₃ - Q₁ Identifikasi outlier

Pandas menyediakan DataFrame untuk manipulasi data tabular yang powerful.

import pandas as pd

df = pd.DataFrame({
    'customer_id': ['C001', 'C002', 'C003'],
    'age': [25, 34, 45],
    'total_charge': [45.50, 28.90, 75.20],
    'churn': [False, True, False]
})

# Cek tipe data
print(df.dtypes)

# Konversi tipe data
df['gender'] = df['gender'].astype('category')
df['age'] = df['age'].astype('int8')

Data Manipulation

# Data Selection
charges = df['total_charge']              # Single column
age_charges = df[['age', 'total_charge']]  # Multiple columns

# Filtering
churned = df[df['churn'] == True]
high_spenders = df[df['total_charge'] > 40]

# Multiple conditions
target = df.query("age > 30 and total_charge > 40")

# Data Transformation
df['charge_per_minute'] = df['total_charge'] / df['total_minutes']

# Aggregation dengan GroupBy
